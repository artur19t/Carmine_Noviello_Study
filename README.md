# STM32 Learning (Carmine Noviello)

Этот репозиторий создан для изучения микроконтроллеров STM32 на основе книги  
**“Mastering STM32” — Carmine Noviello**.

## Описание
В книге в качестве среды разработки используется **Eclipse**, однако в данном репозитории примеры реализуются в другой связке и с иным подходом к работе с проектом.

**STM32CubeMX** используется не как полноценная IDE и не как основной генератор кода, а исключительно как инструмент:
- конфигурации микроконтроллера и периферии
- подключения **CMSIS** и **LL-библиотек**
- генерации базовой инициализации (startup, system, init)

Основная разработка, сборка и прошивка выполняются вручную и контролируются разработчиком.

Используемая связка:
- **VS Code**
- **CMake**
- **STM32CubeMX** (конфигурация и подключение библиотек)
- **LL (Low Layer)**
- **OpenOCD** для прошивки и отладки

## Примечание
Проект развивается постепенно. Некоторые части будут дорабатываться и улучшаться по мере изучения книги и появления доступа к аппаратуре.
